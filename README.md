## 기능

OAuth 2.0을 통한 구글로그인, 실시간 데이터베이스와 연동되는 CRUD, ... (진행중)

### `구글 및 깃허브 로그인`

firebase/auth 의 기능으로 구현합니다.
로그인된 유저의 계정정보를 감지합니다.

### `트윗하기`

트윗을 하고(create), 읽어오며(read), 삭제 및 편집(delete, update)할 수 있습니다.<br />
삭제와 편집은 오로지 트윗한 유저만이 할 수 있습니다.<br />
